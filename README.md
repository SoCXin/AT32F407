### 关键特性

* Cortex®-M4主频最高 240 MHz
* 高达 16MB 外部 SPI Flash memory 扩展
* 4 x USART + 4 x UART
* 2 x CAN 2.0B
* Crystal-less USB2.0全速接口
* 10/100M以太网MAC：专用DMA和4K SRAM，支持IEEE 1588，MII/RMII接口
